### 了解 TypeScript
>TypeScript 是 JavaScript 的一个超集，主要提供了类型系统和对 ES6 的支持。TS 增强了代码的可读性和可维护性。

在命令行中输入以下指令就可以在全局享用 TypeScript 语法啦！

```
npm install -g typescript
```

编译 TypeScript 也很简单 直接执行指令： `tsc 文件名` 。
